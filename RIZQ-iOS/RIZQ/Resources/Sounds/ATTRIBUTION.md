# Sound Asset Attribution

## Current Files (Placeholders)

The two `.caf` files in this folder are **procedurally synthesized placeholders**
generated by `RIZQ-iOS/scripts/generate_sounds.py`:

| File | Description | License |
|------|-------------|---------|
| `bead_tap.caf` | Short tasbih-style wood click (~45 ms) | Generated for this project — no license restrictions |
| `completion_chime.caf` | Soft bell-like chime with non-harmonic partials (~850 ms) | Generated for this project — no license restrictions |

The generator script uses only Python's stdlib (`wave`, `math`) — no third-party
audio is involved, so these files are not subject to any external license.

## Replacing With Curated Recordings

To swap in higher-quality recordings (recommended at some point), drop new
files into this folder using the **exact same filenames**:

- `bead_tap.caf` — keep under ~100 ms; mono; ≥-12 dB peak
- `completion_chime.caf` — keep under ~1 s; mono; ≥-12 dB peak

`SoundPlayer.swift` loads by filename, so no code changes are needed.

Good CC0 sources to search:
- [Pixabay Sounds](https://pixabay.com/sound-effects/) — search "tasbih", "wood click", "soft bell", "meditation bell"
- [Freesound](https://freesound.org/) — filter to **CC0** license only

Add the replacement's license/credit row to the table above.

## Regenerating Placeholders

```sh
python3 RIZQ-iOS/scripts/generate_sounds.py
cd RIZQ-iOS/RIZQ/Resources/Sounds
afconvert -f caff -d LEI16@44100 bead_tap.wav bead_tap.caf
afconvert -f caff -d LEI16@44100 completion_chime.wav completion_chime.caf
rm bead_tap.wav completion_chime.wav
```
