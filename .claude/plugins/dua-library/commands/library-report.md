# Library Report Command

Generate a detailed status report of the RIZQ App dua library.

## Report Generation

Run the following queries to gather data:

### 1. Overview Metrics
```sql
SELECT
  (SELECT COUNT(*) FROM duas) as total_duas,
  (SELECT COUNT(*) FROM journeys) as total_journeys,
  (SELECT COUNT(*) FROM categories) as total_categories,
  (SELECT COUNT(*) FROM collections) as total_collections;
```

### 2. Category Breakdown
```sql
SELECT
  c.name as category,
  c.slug,
  COUNT(d.id) as dua_count,
  COALESCE(SUM(d.xp_value), 0) as total_xp
FROM categories c
LEFT JOIN duas d ON c.id = d.category_id
GROUP BY c.id
ORDER BY dua_count DESC;
```

### 3. Difficulty Distribution
```sql
SELECT
  difficulty,
  COUNT(*) as count,
  ROUND(AVG(xp_value)) as avg_xp,
  MIN(xp_value) as min_xp,
  MAX(xp_value) as max_xp
FROM duas
GROUP BY difficulty;
```

### 4. Journey Summary
```sql
SELECT
  j.emoji,
  j.name,
  COUNT(jd.id) as duas,
  j.estimated_minutes as minutes,
  j.daily_xp as xp,
  CASE WHEN j.is_premium THEN 'ğŸ’' ELSE 'ğŸ†“' END as tier,
  CASE WHEN j.is_featured THEN 'â­' ELSE '' END as featured
FROM journeys j
LEFT JOIN journey_duas jd ON j.id = jd.journey_id
GROUP BY j.id
ORDER BY j.is_featured DESC, j.name;
```

### 5. Time Slot Distribution
```sql
SELECT
  time_slot,
  COUNT(*) as assignments
FROM journey_duas
GROUP BY time_slot;
```

### 6. Recent Additions
```sql
SELECT id, title_en, created_at
FROM duas
ORDER BY created_at DESC
LIMIT 5;
```
